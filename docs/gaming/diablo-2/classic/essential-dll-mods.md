# Essential DLL mods
The cherry on top of the battle.net/DRM/mandatory-online-check-in-every-30-day cake is Blizzard
forbidding DLL modding in Resurrected terms of service. Classic stays winning

DLL mods that I find essential:

* D2GL
* D2fps
* PlugY
* BaseMod
* BH maphack
    * Does not support 1.14 patches
    * [cyhyraethz's fork](https://github.com/cyhyraethz/slashdiablo-maphack-giga) is the only version to not
    crash on me with the dreaded `ACCESS_VIOLATION (c0000005)` error when combined with the above DLLs

!!! warning "DLL loading order"

    All of these DLLs combined together have to be loaded in a particular order; to be written
